# بناء ملف APK للتثبيت اليدوي

## الطريقة الأولى: باستخدام Android Studio

1. افتح المشروع في Android Studio
2. اذهب إلى Build > Build Bundle(s) / APK(s) > Build APK(s)
3. انتظر حتى ينتهي البناء
4. سيظهر مسار ملف APK في إشعار
5. انقل ملف APK إلى هاتفك وقم بتثبيته

## الطريقة الثانية: باستخدام سطر الأوامر

1. افتح موجه الأوامر في مجلد المشروع
2. نفذ الأمر:
   ```
   ./gradlew assembleDebug
   ```
3. سيتم إنشاء ملف APK في:
   `app/build/outputs/apk/debug/app-debug.apk`

## تثبيت APK على الهاتف

1. انقل ملف APK إلى هاتفك (عبر USB أو WhatsApp)
2. في هاتفك، افتح ملف APK
3. قد تحتاج إلى تفعيل "تثبيت من مصادر غير معروفة"
4. اتبع خطوات التثبيت

## ملاحظات هامة

- ملف debug APK يحتاج إلى توقيع تصحيح
- للتوزيع، تحتاج إلى ملف release APK موقّع
- تأكد من تفعيل "مصادر غير معروفة" في إعدادات الهاتف
